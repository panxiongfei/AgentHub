## 💻 开发者快速上手

### 环境准备
```bash
# 1. 设置AI模型API密钥
export MODEL_GEMINI_API_KEY="your_api_key"

# 2. 安装依赖
pip install -r requirements.txt
cd frontend && npm install

# 3. 验证环境
python regression_test.py
```

### 开发规范
- **每次代码变更必须运行回归测试**
- **新功能开发同步添加测试覆盖**  
- **遵循现有的代码风格和架构设计**
- **优先使用项目已有的AI和浏览器工具**

- `regression_test.py` - 主回归测试脚本

- `app/core/ai_file_processor.py` - AI文件处理系统
